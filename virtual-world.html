Virtual World Final Project - 2022

Overview

In a virtual world game, houses and routers were connected to streets and zones. I used high-level code to create objects like:

  • A calculator
  • A Pig Latin translator
  • In-game chat message cubes and multiplication tables

Technologies Used

  • High-level programming
  • Virtual World Game Engine

Challenges Faced

Designing and coding interactive objects while managing the connections between different zones required careful planning and testing.

Outcome

The project successfully combined creativity with logical problem-solving, resulting in a unique and functional virtual environment.
