Space Invaders (Gameboy) - 2023

Overview

I utilized PyGames to develop a 2D game where a ship at the bottom of the screen shoots targets that shift downward while running. I worked with a **Raspberry Pi** to connect the game to a 3D-printed Gameboy and wrote scripts to detect key pressing through GPIO pins to trigger in-game actions.

Technologies Used

  • PyGames
  • SolidWorks
  • Python
  • Raspberry Pi OS

Challenges Faced

Integrating the hardware pins with the game logic was challenging. Ensuring responsive key detection while the game ran smoothly required optimization of GPIO input and output.

Outcome

The game ran seamlessly on a 3D-printed Gameboy. It was an exciting combination of software development and hardware engineering.
